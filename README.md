## 1. 서비스 개요
자주 등장하는 면접 문항에 대한 답변을 준비하고 싶어

여러 사람들 앞에서 말하는 상황을 경험하고 싶어

면접 프로세스를 경험하며 면접 전형 과정에 익숙해지고 싶어

내가 잘 준비하고 있는지 다른 사람들의 피드백이 궁금해...

**혼자 면접을 준비하는 취업준비생들의 든든한 면접 파트너, 온터뷰**
